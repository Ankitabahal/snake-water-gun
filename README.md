 # 🐍 Snake Water Gun – Python Project

This is my *first Python mini-project: a simple **Snake, Water, Gun game*, inspired by [CodeWithHarry](https://www.codewithharry.com/). It's a fun console-based game where the user plays against the computer.

---

## 🔧 What I Learned

- Python basics (variables, conditionals, loops)
- Random module usage
- Logic building
- User input handling
- First experience with GitHub!

---

## 💻 How to Play

1. Run the Python script in your terminal or any Python IDE.
2. Choose one of the following:
   - s for Snake
   - w for Water
   - g for Gun
3. The computer will randomly choose its move.
4. The result will be displayed: Win / Lose / Tie.
